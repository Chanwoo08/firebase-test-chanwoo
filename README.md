# 📚 FocusFlow: 몰입 기반 학습 관리 웹 애플리케이션

FocusFlow는 포모도로 타이머와 체계적인 학습 기록 관리를 결합하여 사용자의 집중력과 학습 효율을 극대화하는 웹 애플리케이션입니다.

## ✨ 주요 기능 (Features)

본 프로젝트는 사용자가 효과적으로 학습 시간을 측정, 기록하고 목표를 달성할 수 있도록 설계된 세 가지 핵심 기능으로 구성되어 있습니다.

### 1. ⏰ 집중 시간 (포모도로 타이머 & DB 기록)

사용자가 설정한 시간 동안 **집중하여 학습**할 수 있도록 돕는 기능입니다.

* **카운트다운 타이머:** 설정된 시간(기본 25분)이 **1초씩 감소**하며 카운트다운을 표시합니다.
* **시간 변경:** 타이머 시작 전, `+` / `-` 버튼을 사용하여 집중 시간을 쉽게 조절할 수 있습니다.
* **데이터베이스 기록:** 타이머가 종료될 경우, 해당 집중 시간(분)이 데이터베이스에 **자동으로 누적 기록**됩니다. 이 누적 시간은 목표 달성 현황 및 주간 통계에 활용됩니다.

---

### 2. 🎯 목표 달성 현황 (Goal Tracker)

일일 학습 목표를 설정하고 현재까지의 진행 상황을 시각적으로 확인하는 기능입니다.

* **일일 목표 시간 설정:**
    * 사용자가 목표 시간을 설정하고 **'설정하기'** 버튼을 누르면 해당 값이 데이터베이스에 저장됩니다. (자동 저장 대신 명시적인 저장 버튼을 사용)
    * `+` / `-` 버튼을 통해 목표 시간을 간편하게 조정할 수 있습니다.
* **누적 공부 시간 표시:** 1번 기능(집중 시간)을 통해 기록된 **누적 시간**이 실시간으로 표시됩니다.
* **달성률 시각화:**
    * `달성률 = (누적 공부 시간 / 일일 목표 시간) × 100%` 공식으로 계산됩니다.
    * 계산된 달성률은 **진행률 표시줄 (Progress Bar)**을 이용하여 직관적으로 시각화됩니다.

---

### 3. 📈 주간 공부량 (Weekly Study Analytics)

일별 학습 기록을 분석하여 사용자의 주간 학습 패턴을 한눈에 파악할 수 있도록 제공합니다.

* **데이터 출처:** 1번 기능(집중 시간)에서 기록된 일별 데이터를 기반으로 합니다.
* **시각화 방식:** **Bar 형식 그래프**를 이용하여 데이터를 표현합니다.
* **축 구성:**
    * **X축:** 요일 (일요일을 시작으로 `일, 월, 화, 수, 목, 금, 토` 순으로 표기)
    * **Y축:** 해당 요일의 총 공부 시간 (분)

---

## 💻 프로젝트 구현 환경

* **프론트엔드:** React, React Router, Framer Motion (애니메이션), Recharts (차트 시각화)
* **상태 관리:** React Context (AuthContext, StudyContext)
* **데이터베이스:** Firebase Firestore (공부 기록, 목표 시간, 과목 정보 저장)
* **인증:** Firebase Authentication (로그인/로그아웃/회원가입)

## 📌 추가 기능 및 사용자 경험 개선

* **로그인 상태별 라우팅:**
    * 로그인 시: 홈 화면의 **'지금 시작하기'** 버튼 클릭 시 **대시보드 (`/main1`)**로 즉시 이동.
    * 로그아웃 시: **'로그아웃'** 버튼 클릭 후 **홈 화면 (`/`)**으로 이동하며, 홈 화면의 버튼이 **'지금 시작하기'**로 변경됨.
* **사용자 인증**: Firebase를 통한 안전한 로그인/회원가입/로그아웃 기능 구현.
